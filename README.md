# dashboard-memo-app

https://yoshidafamily.github.io/dashboard-memo-app/

その日の予定、タスク、メモなどをまとめて閲覧できるアプリ。

## 機能一覧

- （必須）メモの作成
- （必須）作成したメモの閲覧
- （拡張）カテゴリの動的生成
- （拡張）クラウドサービスとの連携

## 実現方式

- WebAppとして開発する。
- GitHub Pagesで公開する。別案として、GASかHeroku。
- データは、ひとまずWebStorageに保管するものとする。
- PWA化して、AndroidAppとしても動くようにする。
- フレームワークは未定。
